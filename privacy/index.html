개인정보 처리방침

너의 눈코입 (Eyes Noses Lips)

최종 수정일: 2026년 3월 31일

1. 수집하는 정보

"너의 눈코입"(이하 "앱")은 다음 정보에 접근합니다:

  • 사진 라이브러리: 아기 사진을 찾고 얼굴을 인식하기 위해 기기의 사진에 접근합니다.
  • 알림: 매일 사진 기록을 놓치지 않도록 알림을 보내기 위해 사용합니다.

2. 정보의 처리 방식

앱은 모든 데이터를 사용자의 기기 내에서만 처리합니다.

  • 얼굴 인식은 Apple Vision 프레임워크를 사용하여 기기 내에서 수행됩니다.
  • 얼굴 특징값(임베딩)은 기기에만 저장되며, 외부 서버로 전송되지 않습니다.
  • 크롭된 얼굴 이미지는 기기의 앱 저장소에만 보관됩니다.
  • 원본 사진은 복사하지 않으며, 참조(식별자)만 저장합니다.

3. 데이터 저장

  • 아기 생년월일, 날짜별 사진 선택 기록, 얼굴 임베딩 데이터가 기기에 저장됩니다.
  • 위젯 기능을 위해 App Group 공유 컨테이너에 데이터가 복사될 수 있습니다.
  • iCloud 백업 기능을 켜면 iCloud에 데이터가 동기화될 수 있습니다.

4. 외부 전송

앱은 어떠한 개인정보도 외부 서버에 전송하지 않습니다.

  • 네트워크 통신을 하지 않습니다.
  • 분석(Analytics), 광고, 추적 도구를 사용하지 않습니다.
  • 제3자에게 데이터를 공유하지 않습니다.

5. 데이터 삭제

설정 → "모든 데이터 초기화"를 통해 앱에 저장된 모든 데이터를 삭제할 수 있습니다. 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.

6. 아동 개인정보

앱은 아기의 사진을 다루지만, 해당 데이터는 기기에만 저장되며 외부로 전송되지 않습니다. 모든 처리는 보호자의 기기 내에서 이루어집니다.
